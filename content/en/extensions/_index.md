## 5,398 Chrome extensions. Scanned without consent.

LinkedIn's JavaScript fingerprinting system probes every visitor's browser for the presence of Chrome extensions. The scan covers browser extensions across every category — from productivity tools and ad blockers to extensions that reveal religious beliefs, political opinions, disability status, and employment-seeking activity.

This database contains every extension LinkedIn is known to scan for, based on analysis of LinkedIn's production JavaScript bundles.
