{{% blocks/lead color="white" %}}

BrowserGate is an open-source project that makes it easy for EU residents to file formal complaints about LinkedIn's practices — from dark patterns and GDPR violations to anti-competitive behaviour.

{{% /blocks/lead %}}

{{% blocks/section type="row text-center h1" %}}

Complaints are generated using AI, formatted as proper legal letters, and sent via Deutsche Post.
